Matematicas de cuarto año de secundaria

Observa a la izquierda la pareja de figuras semejantes, tienen la misma forma pero están representadas con tamaños diferentes, una puede considerarse una ampliación de la otra.

Dos figuras planas se consideran semejantes si existe la misma proporción, llamada razón de semejanza, entre sus lados homólogos y además sus ángulos homólogos son iguales.

Proporciones geométricas y Semejanza de triángulos

AB/A'B'=BC/B'C'=AC/A'C'

Teorema de Seno y Coseno

Teorema de seno

a/sen A = b/sen B

a/sen A = c/sen C

b/sen B = c/sen C

Teorema de Coseno

a2 = b2 + c2 - 2 x b x c x cos A

b2 = a2 + c2 - 2 x a x c x cos B

c2 = a2 + b2 - 2 x a x b x cos C
